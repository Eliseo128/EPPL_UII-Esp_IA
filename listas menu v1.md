A continuación se presentan **60 ejemplos de enunciados diferentes** para practicar un **programa tipo menú con listas**.
Todos parten de **una lista vacía** y usan las opciones:

1. Agregar elemento a la lista
2. Modificar elemento de la lista
3. Borrar elemento de la lista
4. Mostrar elementos de la lista (usando **for**)
5. Salir (usando **break**)

Se asume el uso de **while True, if, elif**, **sin validar datos**, **sin funciones**, **nivel principiante**.
